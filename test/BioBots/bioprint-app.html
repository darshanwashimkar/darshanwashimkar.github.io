Customer usage of BioBot-1

ReadMe

  • Assumptions
    • According to my understanding of this assignment, I had to write a tool to assist analyzing BioBot 1 prints.
    • Because I already got the JSON file, I assumed that I had to write the client-side tool. I might write it differently if I have control over server-side.
    • The information such as "cl_duration" and "cl_intensity" might be too technical/unimportant for the user of this tool. So neglected that.
  • Libraries Used
    • Jquery : https://jquery.com/
    • Bootstrap : http://getbootstrap.com/
    • Tabulator : http://olifolkerd.github.io/tabulator/
    • I also used http://stackoverflow.com/ for the help
  • Usage
    • Tool basically loads the JSON file from : http://darshanwashimkar.github.io/test/BioBots/bioprint-data.json
    • Hovering over the cell shows its value. It also applies for the headers of table.
    • You can search the information with Model number of BioBot-1, Email address of the user.
    • You can also search the data by giving minimum and maximum value for alive and dead cells.
    • To apply all the filter at a time use "Apply All Filters" button.
